Introduction

New
This is an interactive prototype meant for documentation. We wanted to see if it is possible (and feasible) to create a documentation template in the same figma file. In this document we'll be discussing how to use this template.

Quick Start

New
When you duplicate this file from community, select all the elements and copy-paste them into your design file. This way you can also copy the Master Components and use them however you like. You can also apply your own styles on Master Components to make it consistent with your design or leave it as it is.
Title
We will only show an example to showcase this template. Feel free to add your own creativity and share remixes in Figma Community.

Create interactions between Master Component of Navigation and Frames you create. This way you only need to link them once.

Limitations

New

Since this is a Figma prototype and not actual website, it is bound to have some limitations.

  • You cannot copy text in prototype mode which is a big no if you are adding coding blocks. To copy the actual code or any text, they need to go to actual design file and copy the text from there
  • To get actual properties like text properties, padding, icons etc; developers still need to access the design file.
